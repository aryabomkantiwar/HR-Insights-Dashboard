 Project Overview

The **HR Insights Dashboard** is an interactive Power BI project designed to analyze employee data and uncover meaningful workforce insights. It helps HR professionals and business leaders monitor key metrics such as employee attrition, workforce demographics, job satisfaction, and performance trends to support data-driven decision-making.

---

 Project Objectives

- Analyze employee attrition trends
- Monitor workforce demographics
- Track employee performance
- Measure job satisfaction
- Identify factors influencing employee turnover
- Support strategic HR decisions with interactive dashboards
